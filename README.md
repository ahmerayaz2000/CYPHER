**CYPHER** is an ultra-premium, highly interactive, and gamified B2B cybersecurity awareness training academy and subscription e-commerce platform. Built to secure modern remote teams, prevent phishing breaches, and satisfy regulatory audits (SOC2/ISO27001), CYPHER replaces boring compliance slides with a playable interactive sandbox right on the landing page.

---

## 🚀 Live Interactive Sandbox Modules

Visitors can play three gamified cybersecurity training lessons directly on the homepage, accumulating compliance score metrics to claim their verifiable credential:

1. **Phishing Lab**: A fully simulated webmail client where users click on-screen hotspots (typosquatted domains, unencrypted URLs, alarmist generic headers) to analyze threat clues and decide whether to block the email or mark it as safe.
2. **Password Architect**: A real-time strength gauge that calculates brute-force decryption clocks (ranging from Instant to 9.4 Billion Centuries) and validates standard security rules dynamically as users type.
3. **Safe URL Quiz**: A rapid-fire domain quiz mapping safe login coordinates with instant glowing green/red visual response indicators.
4. **Verifiable Certificate Minter**: Completing all lessons unlocks a canvas-drawn certification panel. Typing your name and clicking "Mint" compiles a high-res verifiable PNG certificate complete with digital audit hashes, accompanied by a dynamic confetti canvas celebration.

## 🛒 Flat-Rate B2B E-Commerce Mechanics

CYPHER features a streamlined subscription sales pipeline designed to attract high-intent corporate clients:

## 🛠️ Technology Stack & Optimization

To exceed the **3-second maximum load time** and maintain a perfect Lighthouse score, CYPHER is architected entirely without heavy JavaScript frameworks or hydration latency:

*   **Markup**: Semantic HTML5 containing robust SEO meta tags, social OpenGraph structures, and screen-reader accessibility landmarks.

## 📦 How to Run Locally

Since the project is built on highly optimized vanilla frontend standards, there are no dependencies to install or build steps to compile:

1. Clone this repository directly:
   ```bash
   git clone https://github.com/yourusername/cypher-cybersecurity-academy.git
   ```
2. Open `index.html` in any modern web browser:
   * **Windows**: Double-click `index.html` or run `Start-Process index.html` in PowerShell.
   * **macOS/Linux**: Run `open index.html` in your terminal.
3. Alternatively, serve with a lightweight local server (e.g., Live Server extension in VS Code) for the full experience.
